# Overfreight Decal Indonesia
_Repository Resmi Komunitas Indonesia Overfreight_

---

Selamat datang di gudang ID decal komunitas **Indonesia Overfreight**.

Halaman ini berisi kumpulan ID decal Roblox yang dikumpulkan dari berbagai anggota komunitas. Perlu diketahui bahwa seluruh ID yang tercantum di sini **bukan milik Indonesia Overfreight Community**, melainkan milik masing-masing pembuatnya.

Setiap ID yang terdaftar akan dilengkapi dengan informasi kreator, mencakup **username Roblox** dan **username Discord** sebagai bentuk penghargaan kepada pemilik aslinya.

---

**Cara Akses:**
Gudang ID ini hadir dalam bentuk website, bukan repository langsung atau file `.md`.
Terdiri dari dua halaman utama:

- **Beranda** — Informasi dan deskripsi mengenai gudang ID ini
- **Daftar ID** — Seluruh ID decal beserta informasi kreatornya
